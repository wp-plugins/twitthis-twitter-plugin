== Description ==

The twitthis button easily allows your blog to be retweeted.


Features

* Integrates the ability to retweet your post with one click
* Easily installed
* Allows you to specify when the button should go (e.g. top or bottom)


== Installation ==

Follow the steps below to install the plugin.

1. Upload the twitthis directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/twitthis to configure the button

For Help goto http://twitthis.com/contact_us
